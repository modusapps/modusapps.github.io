Privacy Policy

Effective Date: June 17, 2026

1. Overview

This Chrome extension enables users to send selected content (such as YouTube transcripts, web page text, or highlighted text) to AI chat services for processing.

We are committed to protecting user privacy and minimizing data collection.

2. Information We Access

The extension may access the following data only when the user initiates an action:

  • Selected text on web pages
  • Content of the active tab
  • YouTube video transcripts (when available)
  • Page URL and basic metadata required for processing

3. How Data Is Used

Data is used only to send user-selected content to the chosen AI service for processing, such as summarization, rewriting, or explanation.

No tracking or profiling is performed.

4. Data Storage

  • No user content is permanently stored
  • Local browser storage may be used for preferences (e.g. AI provider, prefixes)
  • No browsing history is collected

5. Data Sharing

We do not sell or share user data. Content is only sent to AI services chosen by the user at the time of use.

6. Permissions Used

  • Active Tab – read page content when requested
  • Scripting – extract or inject content into AI pages
  • Context Menus – enable right-click actions
  • Storage – save user preferences locally
  • Tabs – open AI chat pages

7. Third-Party Services

When using AI providers, data may be sent to:

  • OpenAI (ChatGPT)
  • Anthropic (Claude)
  • Google (Gemini)
  • DeepSeek
  • xAI (Grok)

These services have their own privacy policies and are responsible for how they handle data.

8. Security

Data is transmitted securely over HTTPS. The extension does not run background tracking or hidden data collection.

9. User Control

Users can disable or remove the extension at any time and control what content is shared.

10. Changes

This policy may be updated periodically. The effective date will always reflect the latest version.
